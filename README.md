


# sayYesGame

Did you ever want to ask your loved one a question but felt they might say “no”? Try sayYesGame!

📍 Create your own game link by adding the player’s name and your question, then share it with them.
The funny part? The “No” button runs away when they try to click it 😄

Try it on your desktop: https://thekassdag.github.io/sayYesGame/
